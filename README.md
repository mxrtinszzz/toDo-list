# Tasks ToDo App

Aplicativo simples de lista de tarefas desenvolvido em **React Native** com **Expo**, permitindo adicionar, visualizar e marcar tarefas como concluídas.

## Funcionalidades

* Adicionar novas tarefas.
* Marcar ou desmarcar tarefas como concluídas.
* Lista rolável para visualizar todas as tarefas.
* Interface limpa com cores escuras e detalhes em roxo/lilás.

## Instalação e Execução

1. Clone o repositório:
   `git clone <URL-do-repositório>`
2. Navegue até a pasta do projeto:
   `cd nome-do-projeto`
3. Instale as dependências:
   `npm install` ou `yarn install`
4. Inicie o app:
   `expo start`
5. Abra em um dispositivo físico ou emulador pelo QR code exibido.

## Uso

1. Digite a tarefa no campo de entrada.
2. Clique em **Add** para adicionar à lista.
3. Toque na tarefa para marcar ou desmarcar como concluída.
   Tarefas concluídas aparecem riscadas para melhor acompanhamento.

## Layout e Estilo

* Fundo escuro com elementos em roxo/lilás.
* Campo de entrada e botão alinhados horizontalmente.
* Lista de tarefas com padding e scroll para fácil navegação.
